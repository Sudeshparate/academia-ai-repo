🎓 Academia AI

AI-Powered Learning Platform for Smart Content Transformation

🚀 Overview

Academia AI is a full-stack AI-powered platform designed to transform educational content into more accessible and engaging formats. It leverages modern NLP and large language model (LLM) techniques to automatically generate:

-📄 Concise summaries from videos/text  
-🎧 Audio podcasts for on-the-go learning  
-📝 Dynamic quizzes for knowledge assessment  

The goal is to reduce learning time and improve content accessibility by converting long-form educational material into structured, digestible outputs.

✨ Features     
-🔍 Video/Text Summarization     
Extracts key insights using NLP and transformer-based models   
-🎙️ Podcast Generation   
Converts summarized content into audio format for passive learning   
-🧠 Automated Quiz Generation  
Generates questions based on processed content to enhance retention  
-🌐 Full-Stack Web Interface  
Interactive UI built with React and backend powered by Flask   
-⚡ Efficient Data Processing  
Optimized backend APIs for smooth data flow and response handling

⚙️ Installation & Setup
1. Clone the Repository
git clone https://github.com/Sudeshparate/academia-ai.git
cd academia-ai
2. Backend Setup
cd backend
pip install -r requirements.txt
python app.py
3. Frontend Setup
cd frontend
npm install
npm start
